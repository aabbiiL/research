# 墨西哥现金贷与循环额度客群运营策略

本专题研究墨西哥现金贷与循环额度的客群运营，不以额度参数优化为主体。方案覆盖前置机会识别、运营权益组合、Push/SMS/App/WhatsApp 等渠道编排、Control Group/Holdout 增量评估，以及中国、印尼、菲律宾、巴西等市场的迁移价值。

## 核心边界

报告使用五层框架，避免混淆产品功能与运营触达：

1. Product Capability：模拟、支用、还款、额度恢复与服务入口；
2. Risk/Offer：资格、额度、临额、期限、价格与负担能力；
3. Operational Treatment：提醒、解释、息费券、Cashback、任务与 Referral；
4. Touch Strategy：客群、时点、客户主张、渠道序列、频控与退出；
5. Experiment & Outcome：Control Group、Treatment Group、Holdout、增量利润与风险护栏。
