# Research Reports

本仓库用于维护研究报告及其可阅读成品。报告按专题独立建目录，避免不同研究对象的源文件、HTML 与版本记录混放。

## 目录规范

```text
研究报告/
└── <专题名称>/
    ├── README.md            # 专题入口、范围与文件说明
    ├── <report>.md          # 报告源文件
    ├── <report>.html        # 可直接阅读的成品
    └── VERSION_HISTORY.md   # 版本记录
```

一级目录和专题目录统一使用简洁、稳定的中文名称。报告文件名使用中文；临时文件、生成脚本和 PDF 不纳入仓库。
